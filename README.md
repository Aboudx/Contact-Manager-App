# Contact Manager

![Contact Manager Badge](https://img.shields.io/badge/Contact%20Manager-WinForms%20Desktop%20App-2D89EF?style=for-the-badge&logo=windows11&logoColor=white)

A simple and modern **Windows Forms Contact Manager** application for managing contacts efficiently with an intuitive desktop interface.

## Overview

This project is a **C# WinForms desktop application** designed to help users manage contact information in a clean and organized way.

It provides a user-friendly interface for adding, editing, searching, viewing, and deleting contacts, along with validation and quick access to social links.

## Features

- Add new contacts
- Edit existing contacts
- Delete contacts
- Clear input fields
- Search contacts by name
- Filter contacts through the view dropdown
- Gender selection using radio buttons
- Input validation using `ErrorProvider`
- Total contacts counter
- GitHub and LinkedIn quick links
- Clean and simple WinForms UI
